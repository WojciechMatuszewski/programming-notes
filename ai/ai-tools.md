# AI Tools

- [jina.ai](http://jina.ai/) allows you to copy any website content as markdown.

  - You can use extension in Raycast.

  - You can prepend the url with `r.jina.ai/<https://your_stuff>`

- [pm2](https://pm2.keymetrics.io/docs/usage/process-management/) allows you to assign a log file for each process you run. **This means the LLM can read application logs in real-time**.

  - Taken from [this Reddit post](https://www.reddit.com/r/ClaudeCode/comments/1oivs81/claude_code_is_a_beast_tips_from_6_months_of/)

- With [`GitMCP`](https://gitmcp.io/) you can turn any GitHub repository into an MCP server.

  - This is useful if the library you are using is not indexed by `context7` or similar MCPs.

- [worktrunk.dev](https://worktrunk.dev/) for managing Git wotkrees with custom hooks and Claude Code integration
